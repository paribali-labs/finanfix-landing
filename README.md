# Finanfix Landing Page

Landing page do [finanfix.com.br](https://finanfix.com.br) — inteligência financeira para profissionais independentes.

## Stack

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion (animações)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output em `dist/`.
